# Response Codes

The Limelight API uses the following HTTP codes:

Code | Meaning | Used by
---------- | ------- | --- 
200 | Success. | *
201 | Resource Created. | POST
400 | Bad Request / Data Error. | *
401 | Auth required. | *
403 | Auth failed. | *
404 | Invalid endpoint. | *
405 | Bad method. | *
500 | Internal Server Error | *
503 | Service Unavailable / Under Maintenance | *
